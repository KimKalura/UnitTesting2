# Unit Testing
Testing involves checking whether the result of the application meets specific requirements or not. Here are pieces of code that test methods from classes, that is, the smallest units of code that can be tested.

## Built with

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)

## Prerequisites

For building and running the application you need:
- JDK 1.8 or higher
- Maven 3

## Dependencies

You don't need any additional dependencies. All dependecies related to database management, server management, security management and so on, will be automatically injected by Maven using the pom.xml file located in the root folder of the project.

# Installation

Clone the project

```bash
  git clone https://github.com/KimKalura/UnitTesting2
```
